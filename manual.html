Инструкция по формированию фактических затрат времени

1. Установка Timesheet в Outlook (только при первом заполнении)

Установка Timesheet разлиачется в зависимость от версии Outlook (web,desktop)

Если вы пользуетесь MAC OS или Linux, то необходимо воспользоваться web версией Outlook

Для установки следует использовать следующую ссылку: https://holding-online.ru/timesheet.xml

  • Web версия
  • Desktop версия

2. Заполнение трудозатрат

Для заполнение трудозатрат необходимо:

  • Указать дату заполнения
  • Добавить проекты
  • Заполнить часы
  • Сохранить данные
