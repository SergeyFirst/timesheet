Инструкция по формированию фактических затрат времени

 

Для заполнения трудозатрат за отчетную неделю по проектам в таймшит сотруднику необходимо выполнить следующие действия:

 

1.       При получении письма о необходимости заполнения трудозатрат следует нажать на ссылку «Timesheet» в письме:

 

2.       В открывшемся окне заполнения трудозатрат необходимо нажать на кнопку «Добавить проект»:

 

 

3.       В списке проектов следует выбрать проекты, которыми сотрудник занимался в указанный день и нажать на кнопку «ОК» в конце списка:

 

4.       Для каждого выбранного проекта необходимо заполнить отработанное время. Если часы по проекту потрачены сверх того, что учтено по СКУД, то необходимо установить признак «Сверх нормы». При необходимости можно добавить комментарий, нажав соответствующую кнопку:

 

5.       После заполнения трудозатрат необходимо нажать на кнопку «Сохранить»
